> Конвертируйте экспорт двухфакторных ключей из Яндекс.Ключ в формат, совместимый с Aegis Authenticator.  
> **Вся обработка локально — ваши данные остаются у вас.**

## 🚀 Демо
 
[Live Demo →](https://rostezkiy.github.io/yandex2aegis)

## 📦 Возможности

- ✅ Чтение JSON-экспорта Яндекс.Ключ (включая файлы с расширением `.txt`)
- 🔍 Извлечение всех записей с `techInfo`, начинающихся с `otpauth://totp/`
- 📝 Создание текстового файла `aegis_import.txt` с одной записью на строку
- 🎨 Современный, интуитивно понятный интерфейс
- 📂 Drag & drop или выбор файла через диалог
- ⏳ Индикатор выполнения конвертации
- 💾 Скачивание результата одним кликом
- 🛡️ Полная локальность: JavaScript обрабатывает файл в браузере, без серверных вызовов

## 🔧 Как использовать

1. **Экспортируйте данные из Яндекс.Ключ**
   - Откройте приложение Яндекс.Ключ
   - Перейдите в настройки → Экспорт
   - Сохраните JSON-файл (или TXT с JSON-структурой) на устройство

2. **Откройте приложение Yandex2Aegis**
   - Запустите `index.html` локально или откройте опубликованную версию

3. **Загрузите файл экспорта**
   - Нажмите «Выбрать файл экспорта» или перетащите файл в область загрузки

4. **Нажмите «Конвертировать»**
   - Приложение извлечёт все совместимые записи

5. **Скачайте результат**
   - Нажмите «Скачать aegis_import.txt»

6. **Импортируйте в Aegis**
   - Откройте Aegis Authenticator → Настройки → Импорт **Plain Text** → Выберите скачанный файл
